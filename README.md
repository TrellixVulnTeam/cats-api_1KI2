# cats-api

REST API with NodeJs, Typescript, Express.js and Sequelize with Sqlite3 |CRUD REST API

# JSON Model:

{
    "name": "Alexander VI",
    "age": "13",
    "breed": "Sphynx"
}

# Main Route:
localhost:9000/api/v1

# Endpoints:
